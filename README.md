#  Employee Management System (Lab 9)

## Overview
This project is a simple LAMP stack web application created for INET 4031 Lab 9.  
It demonstrates a working Apache, MySQL, and PHP environment with basic employee management pages.

---

## Features
- Updated **index.html** homepage with new title
- Employee table with sample data
- Buttons for Create, Find, Edit, and Delete
- **findemployee.html** page with search functionality
- Default search value set to "Gure"
- Working navigation between pages

---

## Technologies Used
- Linux (Ubuntu 24.04 VM)
- Apache2 (Web Server)
- MySQL (Database)
- HTML / CSS / JavaScript
